Producto 1 - Red Empresarial (Equipo 4)

Materia: Conmutación y Enrutamiento (TI-203) - ITLA
Plataforma: PNETLab

Descripción general

Este proyecto implementa una red empresarial de mediana escala, segmentada por departamentos mediante VLANs, con alta disponibilidad de gateway (HSRP), redundancia de enlace (EtherChannel y STP), enrutamiento dinámico con OSPFv2 (área única y soporte para multiárea) y control de acceso mediante ACLs.
